Redis简介

Redis 是完全开源的，遵守 BSD 协议，是一个高性能的 key-value 数据库。

Redis 与其他 key - value 缓存产品有以下三个特点：

  • Redis支持数据的持久化，可以将内存中的数据保存在磁盘中，重启的时候可以再次加载进行使用。可以设置每隔几分钟持久化一次或者直接调用持久化命令。Redis是一个中间数据库，持久化时要写在MongoDB或者MySql里。

  • Redis不仅仅支持简单的key-value类型的数据，同时还提供list，set，zset，hash等数据结构的存储。

  • Redis支持数据的备份，即master-slave模式的数据备份。

Redis优势

性能极高 – Redis能读的速度是110000次/s,写的速度是81000次/s 。

丰富的数据类型 – Redis支持二进制案例的 Strings, Lists, Hashes, Sets 及 Ordered Sets 数据类型操作。

原子 – Redis的所有操作都是原子性的，意思就是要么成功执行要么失败完全不执行。单个操作是原子性的。多个操作也支持事务，即原子性，通过MULTI和EXEC指令包起来。（和MySql相同，也是原子性的）

丰富的特性 – Redis还支持 publish/subscribe, 通知, key 过期等等特性。

Key过期：Redis占用内存，而内存总是有限的，如果在一段时间之内（如12小时）某些数据没有读写操作了，就会在内存中删掉。

下载安装

下载地址：

https://github.com/tporadowski/redis/releases，在其中找到msi文件安装，会自动配置好。

双击安装文件，设置好安装位置，并且勾选“Add the redis installation folder to the PATH environment variable”。其余安装默认即可（默认端口6379，不设定最大内存限制）

安装完成后，在Windows Start图标上右键——选择“计算机管理”，在打开的窗口中选择“服务和应用程序”——“服务”，在服务列表中找到“Redis”就表示安装成功。

进入Redis的安装文件夹，双击里面的redis-cli.exe，就可以使用命令行。

下载Redis Desktop Manager

百度网盘：https://pan.baidu.com/s/15xVRpCT8mkP2uT8PoBHT3g 提取码：v727

下载完成后安装运行，在界面下方点击“Connect to Redis Server”，在弹出的窗口中随便起个名字，Host为“127.0.0.1”，Port为“6379”点击链接就可以了。然后可以在软件列表中发现，在默认情况下，Redis在内存中会生成16个数据库，数据库后按整数标记，分别是“db0”~“db15”
